# Zepto_Data_Cleaning
Cleaned and transformed Zepto dataset using advanced SQL queries by managing null values, deduplicating records, formatting inconsistent data, validating entries, and preparing structured data for further business analysis and reporting.

**Project Overview**

This project focuses on cleaning and transforming raw Zepto dataset using SQL to make it analysis-ready. The dataset contained inconsistencies, missing values, duplicate records, and formatting issues that could affect accurate business insights.

The objective was to improve data quality, ensure consistency, and prepare a structured dataset suitable for further analysis and reporting.

**Objectives**

Handle missing and null values

Remove duplicate records

Standardize inconsistent data formats

Correct data entry errors

Validate and clean categorical fields

Optimize dataset structure for analysis

**Tools & Technologies**

SQL (MySQL)

Data Cleaning Techniques

Query Optimization

**Key Cleaning Steps Performed**

Duplicate Removal
Identified duplicate records using GROUP BY and ROW_NUMBER() functions and removed redundant entries.

Handling Missing Values
Replaced NULL values where appropriate and filtered incomplete records when necessary.

Data Standardization

Formatted text fields (consistent casing)

Standardized date formats

Cleaned special characters and extra spaces

Data Validation
Ensured numerical columns contained valid values and corrected inconsistencies in categorical fields.

Data Structuring
Modified column data types and optimized schema for better performance and analysis.
